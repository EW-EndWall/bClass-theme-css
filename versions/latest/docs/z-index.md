# z-index

The z-index CSS property sets the z-order of a positioned element and its descendants or flex and grid items. Overlapping elements with a larger z-index cover those with a smaller one.

.z-auto
.hover:z-auto
.z-0

.z-1
.hover:z-1
.z-2

.z-3

.z-auto,
.hover\:z-auto:hover {
  z-index: auto;
