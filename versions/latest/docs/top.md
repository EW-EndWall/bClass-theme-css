# Top

The top CSS property sets the vertical position of a positioned element. It does not effect non-positioned elements.

- .top-25
- .hover:top-25
- .top-50
- .hover:top-50
- .top-75
- .hover:top-75
- .top-100
- .hover:top-100

```css
.top-25,
.hover\:top-25:hover {
  top: 25%;
}
.top-50,
.hover\:top-50:hover {
  top: 50%;
}
.top-75,
.hover\:top-75:hover {
  top: 75%;
}
.top-100,
.hover\:top-100:hover {
  top: 100%;
}
```
