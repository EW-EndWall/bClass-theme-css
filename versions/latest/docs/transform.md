# Transform Translate

The transform CSS property lets you rotate, scale, skew, or translate an element.

- .translate-x-0
- .hover:translate-x-0
- .translate-x-0.5
- .hover:translate-x-0.5
- .translate-x-1
- .hover:translate-x-1
- .translate-x-1.5
- .hover:translate-x-1.5
- .translate-x-2
- .hover:translate-x-2

```css
.translate-x-0,
.hover\:translate-x-0:hover {
  transform: translateX(0);
}

.hover\:translate-x-0\.5:hover {
```

## Transform Scale

- .hover:zoom
- .hover:zoom-1.5
- .hover:zoom-2

```css
.hover\:zoom,
.hover\:zoom-1\.5,
.hover\:zoom-2 {
  cursor: pointer;
  transition: all 0.1s ease-in;
}
.zoom,
.hover\:zoom:hover {
  transform: scale(1.1);
}
.zoom-1\.5,
.hover\:zoom-1\.5:hover {
  transform: scale(1.5);
}
.zoom-2,
.hover\:zoom-2:hover {
  transform: scale(2);
}
```
