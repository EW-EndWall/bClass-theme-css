# Left

The left CSS property participates in specifying the horizontal position of a positioned element. It has no effect on non-positioned elements.

- .left-25
- .hover:left-25
- .left-50
- .hover:left-50
- .left-75
- .hover:left-75
- .left-100
- .hover:left-100

```css
.left-25,
.hover\:left-25:hover {
  left: 25%;
}
.left-50,
.hover\:left-50:hover {
  left: 50%;
}
.left-75,
.hover\:left-75:hover {
  left: 75%;
}
.left-100,
.hover\:left-100:hover {
  left: 100%;
}
```
