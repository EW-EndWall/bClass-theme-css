# Resize

The resize CSS property sets whether an element is resizable, and if so, in which directions.

- .resize-none
- .resize
- .resize-y
- .resize-x

```css
.resize-none {
  resize: none;
}
.resize {
  resize: both;
}
.resize-y {
  resize: vertical;
}
.resize-x {
  resize: horizontal;
}
```

### example

```html
<div class="container">
  <div class="resize">Hello World !</div>
</div>
```
